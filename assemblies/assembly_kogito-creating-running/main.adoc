// Purpose statement for the assembly
As a developer of business processes and rules, you can use {PRODUCT} business automation to build cloud-native applications with a domain-specific set of services.

[IMPORTANT]
====
{PRODUCT} is currently for Developer Preview only. Developer Preview releases contain features that might not be fully tested and are not in production-ready state. Customers are advised to avoid running the software for production or business-critical workloads. Customers are not able to open support cases against the Developer Preview releases. Red Hat does not commit to fixing any reported issues and the provided features might not be available in future releases.
====

.Prerequisites
* JDK 11 or later is installed.
* Apache Maven 3.6.2 or later is installed.
